# Backgammon Crypto (USDT/USDC)

## Overview
Application de jeu de Backgammon avec système de paris en crypto-monnaies (USDT/USDC).

## Architecture
- **Backend**: Node.js / TypeScript
- **Smart Contracts**: Solidity (Gestion des paris et escrow)
- **Frontend**: React / Dashboard Integration
- **Tests**: Jest

## Structure du Projet
- `src/`: Code source backend/logic
- `contracts/`: Smart contracts Solidity
- `tests/`: Tests unitaires et d'intégration
- `docs/`: Documentation technique et commerciale
